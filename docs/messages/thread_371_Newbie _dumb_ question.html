Re: Newbie "dumb" question

From: bcostley (bobbyeubanks@yahoo.com) | Date: 2003-06-30 18:28:58 UTC
There isn't a distance that is too far. Of course remote boxes get
fewer visitors. Some folks, including me, really appreciate the more
remote boxes. Some folks, excluding me, even have time to hunt
them. :)

Orphan boxes - those not near where you get back to regularly - are
fine. It is a good idea to note that it is an orphan in the clues so
folks can make an extra effort to bring replacement baggies just in
case repair needs to be done. Log books don't need a lot of pages to
last a while (at least in the PNW). A 50 page log book can last a
long time even for popular/easy to get boxes and would probably
suffice virtually forever in a remote box.

From: (RiskyNil@pocketmail.com) | Date: 2003-07-01 11:28:29 UTC-07:00
The box I hid that requires the longest hike in Oregon--as far I know--comes in at around ten miles. I've hiked longer distances to hide a box, but there are trailheads closer to the box than the one I started at.

There are boxes I've placed on the AT that might require 20 mile hikes to reach, but I also don't know if there's a closer trailhead one might be able to reach it from, so who knows?

As for that box that requires a multi-day rafting trip to reach--it is possible to get it on a day hike from the nearest trailhead. I did, and it's only about 24 miles (round trip).

I still don't know of any boxes that can't be done in a single day, though, if you're up for it. =)
