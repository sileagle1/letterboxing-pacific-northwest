There's not much you can most likely do right now, but you might want to make a mental note to check on them next summer.

Recent flooding has altered much of the landscape that is located in proximity to water. Sunshine Point Park, for instance, is almost entirely gone now.

Somebody I work with has been doing a lot of work in the area (actually, a lot of somebodies...). This PDF shows some of the changes in the Mt. Rainier area:

http://www.angelfire.com/dragon/cliomouse/Rainierflood06PP.pdf

If you have boxes on Rainier, or if you're just interested in seeing some impressive photos, go check it out.

This can also serve as a lesson in placing boxes. Even if it seems you're quite a ways from water, you might not be all the time!
