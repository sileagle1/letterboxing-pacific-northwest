Re: [LbPNW] box updates - etiquitte for MIA -are we pulling wings off a dead fly?

From: mudstumpers (mudstumpers@yahoo.com) | Date: 2003-06-10 12:19:17 UTC-07:00
Does anyone else think that if the placer has abandoned a box and no-one else takes on the stamp replacement or care for it as an orphan, that the box clues should be removed or at least a notation made on the clue page, that it has gone missing?

If not completely removed, then list it as missing on the main page's state listing so we don't waste time hunting for clues for something that isn't even there and hasn't been there for a long time.

With the new site, it seems like there should be a way to update each of the clue pages to include the " reported MIA as of DDMMYY" to be added if it is verified missing. Like Oxbow #2 and Council Crest and Rock Creek, every time a new boxer goes on a hunt for these permanently missing boxes, the thread starts again.

It would make LB outings much more pleasant and less frustrating to hunt for a box that is known to be alive and well as of DDMMYY, rather than finding out after you get home from a tedious search that it has been reported missing several times, even a year ago and worse tedium is hunting through the banter on the threads to see if there is a mention of each and every box that you plan to find.

It would be especially frustrating for someone from out of state on a 1 time short visit. You could schedule your time to search only for boxes that are alive and well rather than come back empty handed. Granted it is nice to go to the areas even if there isn't a box, but if your plan is to find a box and you spend time hunting for something has been missing ..... Especially if you spend time deciphering cryptograms like Rock Creek and come up emptyhanded!

Just a thought, or a fly in the ointment.

Re: box updates - etiquitte for MIA -are we pulling wings off a dead fly?

From: bcostley (bobbyeubanks@yahoo.com) | Date: 2003-06-10 19:53:26 UTC
I've gotten as frustrated as you looking for what I later found were
missing boxes. But there are problems with trying to establish a
status system. Check the main list archives from about a year ago. I
managed to start a pretty nasty thread when I suggested the
posssibility of a status process.

Some examples of potential problems:

Had Funhog disappeared I'm sure that Hidden Dragon would have been
in danger of being "delisted" last year. It took me three tries to
find. Better letterboxers than me have also failed to find it.

You might think that a rigirous verification sysytem, like a
previous finder verifying a box gone would work. But strange things
can happen. For example Cedar Hills Twin #1 was gone for a long
time, maybe even a year, and then mysteriously reappeared.

Maybe a notation of "not maintained" would be better. Then folks
would know to surf the archives for info.

But since lots of folks think that missing boxes are just part of
the fun I doubt there will ever be a consensus built to handle
things differently.
