Warning! Don't use GPS in Washington State Parks

From: dbltall42 (dbltall@gmail.com) | Date: 2007-09-07 19:51:52 UTC
A letterbox of mine was pulled from a Washington State park, and I
wanted to alert people.

I want to warn all users of Washington State Parks - do not visibly
use anything that might look like a GPS to a ranger. He apparently saw
someone using what he thought was a GPS, (unlikely for this box - it
was probably one of those PDAs people use for their clues - someone
who still thinks my letterbox is a geocache is obviously not one for
the finer distinctions) and followed them so he could see where the
"geocache" was.

So anyone using a GPS, or anything a park ranger might think is a GPS,
in a Washington State Park, even for legitimate reasons (such as
wanting to be able to find your way back to the parking lot), runs the
risk of being followed surreptitiously by a ranger looking for geocaches.

i asked why there were no signs posted prohibiting geocaching, and he
said that it is posted on "the geocaching site" so I should have seen
it there, and then added

"Yes, you are correct, there is no "sign" but if we put up signs for
everything illegal, we would have to cut down all the trees and put
signs up everywhere. "

Mariette

Geocaches and letterboxes are not prohibited in Washington State
Parks, but do require permits. This is covered by this directive -

http://www.parks.wa.gov/downloads/Directive%2006-01%20Geocaching%20%20letterboxing%20and%20related%20activities.doc

If you search for 'washington state parks letterboxing' on Google it
is the first hit.

Letterboxes are lumped together with geocaches by the Deputy Director
and all are referred to as 'caches' at times in the directive so don't
be confused. If you read the entire document it's clear that the rules
apply to all versions of our game of finding hidden containers in the
woods.

You letterbox wasn't pulled because geocaches are illegal and it was
mistaken for one, it was pulled because you are required to get a
permit to place it but you have not done so.
