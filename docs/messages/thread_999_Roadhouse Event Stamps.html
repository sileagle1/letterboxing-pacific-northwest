I have to pipe in to say that Brett's Roadhouse event stamps are,
indeed, wonderfully carved. And posting them to the files would be up
to him. I do, however, disagree that either of them should become
travelers.

They are very special stamps that commemorate this event only, so
passing it around to the rest of the world would cause it to lose its
specialness (is that a word). If you want the event stamp you gotta
show up at the event. That's how I always thought it was supposed to
go.

Maybe he can create another stamp that can hit the road for other
events, but the Roadhouse Gathering Event Stamps should now be retired,
in my opinion.

Best regards,

Amanda Briles
The Paisley Orca
