I'm not sure how long this box will last since it *is* technically in
plain view. But I'm hoping it's out of the way enough that everyone
that happens to walk near it overlooks it. *fingers crossed* It's
HARD finding a good hiding place in downtown Seattle! A total lack
of parks, trees, and rocks to hide boxes. It's an embarassment! =)

Anyhow, if you want to be sure of getting the box, sooner is probably
better than later. If you have some black electrical tape or
something, feel free to cover the box with it to help hide it
better! (The letterbox is purple, but black or some other dark color
might help camoflouge it a bit better.) The next time I'm back
there, I plan to do just that, but it might be awhile before I can
make it back again.

http://www.geocities.com/rscarpen/Lb/US/WA/SeattleWT

Enjoy! =)
