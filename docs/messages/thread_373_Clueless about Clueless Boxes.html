From: (RiskyNil@pocketmail.com) | Date: 2003-07-01 11:28:30 UTC-07:00
If you ask me, there is NO SUCH THING as a truly clueless box. If it uses pictures instead of words, that picture is still a clue. If the location is in the title of the box, that's still a clue. If a clue isn't written down but told verbally, it's still a clue.

At best, you can make the clue as short as possible, but eventually, something needs to tell someone where to look if you want people to find it.

-- Ryan
