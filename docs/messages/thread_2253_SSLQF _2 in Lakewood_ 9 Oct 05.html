This is a cross-post from the Washington discussion board at Atlas
Quest reference the event map/directions found in the photo section
(soon to be found in the photo section of LbPNW, as well).

Hey, I noticed on the event 'map' photos in Atlas Quest that I wasn't
able to read the text on the images. GT mentioned to follow
the 'cascade' of photos down to the largest size and then you should be
able to read them. I did that but still the text was illegible. But I
noticed by clicking on the Explorer Toolbar at the top of the
window: 'View', then 'Full Size' it would pull up the photo to the
point I could read all of the text clearly.

If you run into this problem, perhaps this work around may help?
