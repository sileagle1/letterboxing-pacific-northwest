From: doubletreepfx (greenthummm@hotmail.com) | Date: 2007-12-09 16:22:01 UTC
An impromptu trip to McIver State Park is always good for a few
boxes. Who could possibly have them all? So I was desperate
yesterday since I hadn't found a box for several weeks, and Milo
McIver Park near Estacada sounded like a great place to go.

While searching for Protection, I instead got bit by a Chigger. What
an unexpected buggy surprise. Very nice carving Catbead, if you read
this post. Thanks to Maiden for the nice box. It's in fine
condition.

As are Mi(lo) or Bust by Funhog, and the Clackamas River box by
Paisley Orca. Thanks both of you for some cool images and a fun
afternoon getting some exercise.

Now back to the extensive list of chores that I should have been
working on all day yesterday.....
