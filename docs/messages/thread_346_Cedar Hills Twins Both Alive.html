There's been some recent chatter on these boxes. As previously
reported by Funhog, #1 has indeed returned from its mysterious
absence. If you looked for this box last year and didn't find it try
again.

Box #2 is still there though all the horror stories are true. I
guess I didn't step on a snake or get stung by mystery bugs, but my
legs are pretty scratched up. In any case this box is definately not
for children or even the casual letterboxer. It might be best to
wait for fall to try for it. In any case #2 is extremely unlikely to
go missing; I can't think of a box I've found that is less likely to
go missing.
