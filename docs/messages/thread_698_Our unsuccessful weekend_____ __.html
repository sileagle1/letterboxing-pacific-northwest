From: nikanderfamily (blancoandyou@yahoo.com) | Date: 2003-09-15 16:53:09 UTC
We have been fairly lucky in our finds since our finds have been 22
to not finding 5. We are seeing a trend that we have been unable to
find any of DVN2RCKR's letterboxes which we looked for 2 of them this
weekend. The Cape dissappointment one we looked for 1 1/2 hours and
finally called it quits when the kids were whining so bad to go home.
Then we tried to find the Picea Stitchensis one, but had a later
start so it was almost dark and again couldn't find it.And we didn't
get to the Ft Canby one this time to try a second time to find it. It
is very challenging and we love the challenge, but if there is any
help out there we would love it. It is going to drive us crazy until
we can take the time to go find them again which looks to be at least
3 months, to our unhappiness :( Oh well I think those that we don't
find are the ones that make the find more worth it. We hope all of
you had a fun weekend letterboxing. We love to hear the stories.
