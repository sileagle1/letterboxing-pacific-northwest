From: Pungent Bob (PungentBob@HotPOP.com) | Date: 2005-01-16 05:42:21 UTC

The clues to this box are so hard I don't expect anyone to ever find
it. I'm so sure that I've placed an actual one hundred dollar bill
in the box as a first finders reward.

You will need internet access mid-search. A useful URL for those
with wireless internet is: http://www.wififreespot.com/ You can find
a couple of free access spots near your search.

The clues begin with the 100bill.jpg found in the photos section of
the LbPNW Yahoo group at: http://groups.yahoo.com/group/LbPNW/

Good luck!
