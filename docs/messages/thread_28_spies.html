Re: spies

From: rscarpen (RiskyNil@pocketmail.com) | Date: 2003-04-01 05:17:14 UTC
> Oh Chief Fruit Loop, did you disable the members list for this
> group? I could have sworn I saw it before...

That was one of the first things I decided to disable. I figured if
some people want to be anonymous lurkers, they should have the right
to be one! =) It's unlikely you saw any other well-known "spies"
before I disabled it, since it was one of the first things I
disabled. I don't think there *were* any other members besides
myself at the time I disabled it!

Be that as it may, I never figured the members list was particularly
useful. I've never had any trouble finding contact info for someone
I wanted to contact--either on the clue, on their website, or a reply
to one of their posts to the board.

But actually, the main reason I disabled it was because I'm
paranoid. =) Having gone to school majoring in computer science and
working in the industry (yes, I used to work!!!!), I learned two
things about software:

1. The more features and extras that are enabled, the less secure a
system becomes.

2. The more features and extras that are available, the more it
intimidates those that aren't especially computer savy! If you've
ever wondered why "user-friendly" isn't, that's probably the number
one reason!

Now #1 isn't a very big problem here--top notch security on a board
that anyone is allowed to join is kind of silly. But still, why make
it easier for a spammer to join and send all sorts of junk to
everyone? If they really want to do it, they'll still be able to,
but maybe they'd go to a different board instead where they wouldn't
have to jump through as many hoops to get the same thing done?

But #2 I felt was a very valid concern, since not everyone that
enjoys letterboxing is a computer scientist. So I disabled anything
that I thought nobody would miss.

I also wanted to delete the polling feature since I've never
personally created a poll and felt it would be easy to live without,
but some people seem to love it for planning events, so I left it
in. While I didn't have much use for it, I knew others might.

Let's vote on it--if you think that feature should be kept, say yes!

That's a joke, people. *rolling eyes* =) Even though I think we'd
live fine without it, I don't see any harm in leaving it there.

But the one feature I REALLY wanted to disable was the Files
section. I see that feature as a ticking time bomb just waiting to
cause trouble. Probably because someone creates a clue using
Microsoft Word and innocently uploads it not realizing it's been
infected with a virus or some other similar senario.

Ultimately, I decided to leave the feature in mostly because it
appears that it's inexplicably tied to the photos feature, which IS a
terrific feature! But I can't get rid of one without getting rid of
the other. I love seeing pictures of events and signature stamps,
and whatever other pictures people upload. I can't belive the darned
idiots that programed this system set it up that way. Irresponsible!

So anyhow, if you ever do decide to download any files--even if it's
been put there by someone you know and love--it would probably be
best to assume the worst and at the very least run a virus checker
over it. Yahoo Groups might do that automatically--I'm not sure--but
even if they do, a second check wouldn't hurt.

In any case, if nobody ever makes use of the Files feature, that
would be perfectly okay by me! But I do hope the Photos section gets
lots of action. =)

Hmm.... I seem to have strayed quite a bit from the original subject
of disable the members list. I disabled it thinking it would rarely
be useful for anyone, but I would reconsider enabling it if there's a
large demand for the feature. Are there many people out there who
are dieing to have that feature available?

-- Ryan
