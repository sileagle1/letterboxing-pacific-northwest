As far as Beacon Rock...I would be VERY surprised if this box were
missing. You'd never stumble on it by accident, that's for sure. It's
a challenge and a wonderful, well-worth it box once you find it. But be
discreet this trail is so unbelievably busy and it is a difficult box to
get without someone coming across you when you're either getting it,
stamping in and replacing it. Don't give up. I have a feeling it's
right where Ryan put it originally.

Best regards,
