Has anyone found these boxes recently? I haven't heard from the
placers in response to my "contact the placer" messages. None of the
placers of these boxes has many boxes out, and I fear they possibley
aren't participating any longer. I didn't feel comfortable rummaging
about in the bushes for Hidden Greenery in this neighborhood, although
the location was quite easy to find; ditto for Checkerbox. Might try
again if I know someone has found them. When I'm alone I don't do the
same amount of rummaging as I might with a friend. I was able to find
#1 and #2 of Jungle Animal Series but not the rest - has anyone found
the last three lately? Again, no response from my query using
"contact the placer." When one logs the find at LBNA there is no
option to just log the boxes one finds - if you log a find it just
shows up as "found." The log book for #1 is almost full, but there is
a fair amount of room left in #2.
