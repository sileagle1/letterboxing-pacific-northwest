From: Clio Mouse (cliomouse@yahoo.com) | Date: 2005-05-03 12:58:42 UTC-07:00
When I did block carving in junior high, we worked with lino block that's a lot tougher than the stuff used for letterbox stamps. It was a very tough, rust-colored rubber about an eighth of an inch thick, mounted on a sort of burlap weave. We heated it with an iron, even, to make it a little easier to carve. With something so tough to work with, it was easier (even with the same, reasonably safe tools I have now) for the tool to suddenly come loose, overshoot, and gouge a little "v" shape in your hand. If she doesn't know about making little stamps like we do, and more familiar with print-making with that kind of lino block, I can see why she would be concerned. Carving something as soft as an eraser is definitely easier! If she still worries, not much can be done...

ClioMouse
