For some time it's been bothering me that Hood River County is listed
as part of North Central Oregon on the LbNA site. This ends up with
boxes around Cascade Locks being listed alongside those in Bend,
Prineville, etc. Since much of this county is part of the Columbia
River Gorge and is readily accessible to Portland via I-84, it seems
to me that it should be listed as part of the Portland Area page.

I have no idea if this can even be changed but I'd like to know what
others think about it before I hassle Wes. Does anyone besides me give
a rip? Funhog
