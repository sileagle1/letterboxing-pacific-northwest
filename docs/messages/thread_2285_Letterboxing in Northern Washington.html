Buenos dias, La Senora de Fuegos!

Okay, that roughly translates to "Hello, Fire Lady", but that's close
enough, right? =)

Anyhow, if you use the Trip Planner at
http://www.atlasquest.com/lboxes/trip/ you'll be able to search for
all boxes listed on Atlas Quest that match your request. Change the
maximum search distance to 5 miles, set the start of the search to
Portland, and the end of the search to Bellingham, check the box to
only return "drive-bys", and search. It'll list 23 different sets of
boxes that might be what you're looking for. =)

If you don't check the "drive-by" option, it'll return over a hundred
possible boxes you might look for. Not all boxes that are drive-bys
are actually listed as drive-bys, so many of those boxes will fit what
you're looking for, but you will have to weed through them a bit to
make sure it's actually a drive by.

Happy trails! =)

-- Ryan
