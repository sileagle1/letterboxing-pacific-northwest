Questing in Forest Park!

Agency: Friends of Forest Park

Description: Quest in Forest Park! Help create a treasure hunt within the trails of
Forest Park, the country's largest naturally forested urban park. Volunteers will walk a
trail, creating a map of landmarks along the way. Volunteers will return to
headquarters bringing ideas of how to quest or create a treasure hunt on the trail
using natural clues and even riddles. There are several trails throughout the park that
will be part of the end result: a treasure map to be published for park visitors. Be a
part of this exciting adventure! This project is perfect for families kids can help
create memorable riddles using the crooked trees and sword ferns that they
encounter along the way just to name a few!

Impact Area: Environment: Conservation, Children & Youth: Enrichment and
Recreation
